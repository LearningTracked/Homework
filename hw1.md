# 交作業流程

1. 將專案fork一份

2. clone到本地

   ```
   git clone url
   ```

3. 新增一個branch

   ```
   git branch new-branch
   ```

4. 切換到新branch

   ```
   git checkout new-branch
   ```

5. 寫作業

6. 有新增檔案的話要add

   ```
   git add .
   ```

7. commit

   ```
   git commit -am "done"
   ```

8. push到遠端

   ```
   git push -u origin new-branch
   ```

9. 發出pull request

10. 待確認後merge 

11. 刪除branch
